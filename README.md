## 数据来源

- **官方数据源**: [国家统计局 - 70个大中城市商品住宅销售价格变动情况](https://www.stats.gov.cn/)
- **时间范围**: 2022年5月 - 2025年10月（42个月）
- **数据类型**: 新建商品住宅 & 二手住宅价格指数（环比、同比）
- **覆盖城市**: 完整的70个大中城市数据

> 📍 **Demo展示以广州为例**，但项目已保存全部70城数据（2022.5至今）。
>
> 你可以在代码中轻松切换为自己的城市，一起见证你的"财富神话"！

**声明**: 本项目仅供个人学习和参考使用，数据来源于公开渠道。

## 累计涨跌幅计算说明

使用**环比(MoM)复利累乘**算法 - 就像滚雪球一样,只不过是往下滚:

```
累计指数 = 100 × (第1月MoM/100) × (第2月MoM/100) × ... × (第N月MoM/100)
累计涨跌幅 = 累计指数 - 100
```

> 例：连续2个月环比分别为99.5和99.0，累计指数 = 100 × 0.995 × 0.99 = 98.505
>
> **恭喜！您的资产成功"优化"了1.495%** 🎉

详细计算过程见浏览器控制台日志 - 数学不会骗人。
