# URL Verification Plan

## 问题发现
部分URL是基于模式推断的，但文章ID（最后的数字部分）无法预测，需要逐一验证。

## 验证策略

### Method 1: 逐个搜索验证（推荐）
对dataSource.json中的每个URL进行HTTP请求验证：
- 200 OK → URL正确
- 301/302 → 可能需要更新为新URL
- 404 → 需要重新搜索正确URL

### Method 2: 批量搜索获取
使用Web搜索为每个月份单独查询：
```
"2022年5月份70个大中城市商品住宅销售价格变动情况" site:stats.gov.cn
"2022年6月份70个大中城市商品住宅销售价格变动情况" site:stats.gov.cn
...
```
